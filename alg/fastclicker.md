### Fast Clicker
```
Area -
        Label (прямоугольник с надписью)
        Image (прямоугольник с картинкой)

Label, Image - наследники класса Area
```
```
Класс Area:
    конструктор(т.е. __init__): 
        rect объект
        color_fill
        окно, на котором рисуем
    методы:
        нарисовать
```
```py
# пусть класс Label рисует всегда одним и тем же шрифтом, кеглем и цветом
# по желанию можете сделать универсально
```
```
Класс Label(наследник класса Area):
    конструктор(т.е. __init__):
        свойство - цвет текста
        свойство - текст
        свойство - объект Font(шрифт, кегль)

    методы:
        нарисовать (в отличие от Area рисует не только прямоугольник, но и текст на нем)
            отрендерить текс
            отобразить на экране
```
```
Отрисовка n количества карточек:
1 - создание пустого списка карточек
2 - count_cards = 4 (переменная, которая будет хранить количество карточек)
3 - задать значения X,Y - начало первой карточки
4 - цикл for
    - добавление карточки в список
    - задать текст 
    - увеличить X на 100
```
